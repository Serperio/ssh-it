¿Qué es SSH-IT?


SSH-IT es una página web diseñada para aprender Linux y el protocolo SSH para uso remoto. Esta Nace con el propósito de que gente inexperta se adentre en lo que es Linux y SSH, dando las herramientas para que esto no sea complejo ni engorroso. Esta posee un servidor de SSH propio con múltiples ejercicios para formar un aprendizaje didáctico donde tú mismo vas probando los comandos y sus características. Este es el proyecto final del equipo PaKeDi para la asignatura "Introducción a la Ingeniería".
