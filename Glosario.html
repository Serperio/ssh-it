TIP! : Las opciones de los comandos se pueden agrupar, de manera que es lo mismo poner ls -al a ls -a -l.
TIP! : "-" se lee como "./-".
TIP! : El comando "pwd" sirve para ver la ruta donde uno se encuentra.
TIP! : Para especificar el tamaño de un archivo:
  • 1c = 1 Byte
  • 1k = 1 Kilobyte
  • 1M = 1 Megabyte
  • 1G = 1 Gigabyte

Comandos Básicos



Ctrl + "letra" : Atajos básicos para el terminal.

  • Ctrl + a : Lleva el cursor al inicio de la línea de comandos.
  • Ctrl + e : Lleva el cursor al final de la línea de comandos.
  • Ctrl + l : Limpia toda la terminal, similar a lo que hace el comando clear.
  • Ctrl + u : Limpia desde la posición del cursor hasta el inicio de la línea. Si se está al final limpia la línea entera.
  • Ctrl + k : Limpia desde la posición del cursor hasta el final de la línea. Si se está al inicio limpia la línea entera.
  • Ctrl + h : Hace lo mismo que la tecla backspace, borra el caracter inmediatamente anterior a la posición del cursor.
  • Ctrl + w : Borra la palabra inmediatamente antes del cursor, si está al medio de una palabra borrara desde el cursor hasta el inicio de la palabra.
  • Alt + d o Esc + d : Borra la palabra siguiente después del cursor, si está al medio de una palabra borrara desde el cursor hasta el final de la palabra.
  • Ctrl + p : Establece la línea de comandos con el último comando introducido.
  • Ctrl + r : Inicia la búsqueda de comandos usados anteriormente, tecleando parte de un comando usos anteriores que hayamos realizado incluyendo las opciones y parámetros. Hecha una búsqueda pulsando de nuevo la combinación de teclas encontraremos coincidencias anteriores.
  • Ctrl + c : Termina el proceso que se esté ejecutando, útil para recuperar el control del sistema.
  • Ctrl + d : Sale de la terminal, similar al comando exit.
  • Ctrl + z : Suspende la ejecución del proceso que se está ejecutando y lo pone en segundo plano, con el comando fg podremos volver a continuar su ejecución.
  • Ctrl + t : Intercambia la posición de los dos caracteres antes del cursor, útil para corregir malos tecleos.
  • Alt + f : Mueve el cursor al final de la palabra donde se encuentra el cursor, si se ejecuta dos veces este va al final de la siguiente palabra.
  • Alt + b : Mueve el cursor al inicio de la palabra donde se encuentra el cursor, si se ejecuta dos veces este va al inicio de la anterior palabra.
  • Tab : Autocompleta comandos o rutas de directorios o archivos.


ssh : Establece una conexión de tipo SSH a una terminal.

  • ssh [usuario@dominio]
  • ssh ["nombre_usuario@dominio"] -p [numero_puerto] : Inicia sesión en un puerto específico.
  • ssh -i ["nombre_key"] ["nombre_usuario@dominio"] : Se utiliza para autentificar una key pública.


man : Muestra el manual del respectivo comando.

  • man [comando]


cd : Cambia el directorio por defecto, para así navegar en los distintos directorios.

  • cd [opciones] [nombre_directorio]
  • cd ./"directorio"/ : Ir al directorio seleccionado. Solo se puede acceder a este si es un sub directorio de la ruta en la que uno se encuentra.
  • cd -L : Salir del directorio.
  • cd ~ : Directorio de inicio.
  • cd - : Último directorio en el que estabas.
  • cd .. : Subir al directorio superior.
  • cd –P : Usar la estructura del directorio físico sin seguir enlaces simbólicos.


ls : Mostrar archivos visibles del directorio.

  • ls [opciones]
  • ls -al : Muestra todos los directorios.
  • ls -a : Lista todos los archivos, incluyendo los root.
  • ls -l : Lista los archivos visibles con mucho más detalle, especificando sus permisos, el número de enlaces rígidos, el nombre del propietario, el grupo al que pertenece, el tamaño en bytes, y la fecha de modificación.
  • ls -r : Invierte el orden de listado de los archivos.
  • ls -d : Lista el nombre del directorio en vez de los archivos que hay en él.
  • ls -s : Muestra delante del nombre del fichero el tamaño en kilobytes del mismo.
  • ls -S : Ordena el listado por tamaño de los archivos.
  • ls -1 : Muestra el listado en una sola columna.
  • ls -m : Muestra los archivos en una línea y separados por comas.


whoami : Imprime el nombre del usuario que lo usa.

cat : Lee los archivos y muestra su contenido.

  • cat [opciones] [nombre_archivo]
  • cat [nombre_archivo] : Lee el archivo especifico
  • cat > [nombre_archivo] : Se crea un archivo en el directorio establecido y se puede comenzar a escribir en él. Termina de escribir con Ctrl + d. Si el nombre del archivo ya existe este se sobrescribe.
  • cat >> [nombre_archivo] : Escribe al final de un archivo especifico.
  • cat -n [nombre_archivo] : Lee las lineas con su respectivo número de línea.
  • cat -b [nombre_archivo] : Lee solo líneas no vacías con su respectivo número de línea.
  • cat -E [nombre_archivo] : Marca el final de cada línea con un $.
  • tac [nombre_archivo] : Lee el archivo especifico en orden inverso.
  • cat -v [nombre_archivo] : Muestra caracteres no imprimibles.
  • cat -s [nombre_archivo] : Lee el archivo y suprime las líneas vacías repetidas.


pwd : Muestra la ruta en la que estás.

file : Muestra el tipo y formato de un archivo.

  • file [opciones] [nombre_archivo]
  • file [nombre_archivo] : Muestra el tipo de archivo con su respectivo nombre.
  • file * : Muestra el tipo de todos los archivos en donde uno se encuentra con sus respectivos nombres.
  • file ./[nombre_directorio]/* : Muestra el tipo de todos los archivos de la ruta establecida con sus respectivos nombres. Solo se puede acceder a este si es un sub directorio de la ruta en la que uno se encuentra.
  • file -b [nombre_archivo] : Muestra el tipo de archivo sin su nombre.
  • file --mime-type [nombre_archivo] : Para comprobar que el tipo de fichero sea el que declara ser.
  • file --mime-encoding [nombre_archivo] : Averiguar la codificación de caracteres con que fue guardado un archivo.
  • file -z [nombre_archivo] : Examina los archivos comprimidos.
  • file -p [nombre_archivo] : Conserva la fecha del ultimo acceso.


rm : Se utiliza para borrar ficheros y/o directorios.

  • rm [opciones] [nombre_archivo]
  • rm [nombre_archivo] : Borra el archivo establecido.
  • rm -i [nombre_archivo] : Pide confirmación al usuario antes de borrar el fichero.
  • rm -f [nombre_archivo] : Elimina sin restricciones. No pide confirmación ni produce error si un fichero no existe.
  • rm -r [nombre_directorio] : Borra el directorio establecido con todos sus sub-directorios y archivos.
  • rm -v [nombre_archivo] : Borra el archivo y junto a ello tipea el nombre del archivo que se borró.


find : Busca archivos que cumplan ciertas condiciones dentro de una jerarquía de directorios.

  • find [opciones] [nombre_directorio]
  • find [nombre archivo] : Localiza si el archivo existe en la ruta establecida, si es asi este tipeara el nombre del archivo.
  • find ./[nombre_directorio]/ : Muestra en vertical y jerarquico todos los archivos y subdirectorios que contenga este.
  • find -name [nombre archivo] : Localiza si el archivo existe en directorios o sub directorios de la ruta establecida, si es asi se tipeara la localización de este.
  • find -type d : Busca todos los directorios y sub directorios que se encuentran en la ruta.
  • find -type f : Busca todos los archivos que se encuentran en la ruta.
  • find -type l : Busca todos los enlaces simbolicos que se encuentran en la ruta.
  • find -type c : Busca todos los dispositivos de caracteres que se encuentran en la ruta.
  • find -size [tamaño_archivo] : Busca todos los archivos del tamaño especificado que se encuentran en la ruta. Se antepone un + o - para especificar un "mayor a" o "menor a".
  • find -user [nombre_usuario] : Busca ficheros que pertenezcan al usuario "nombre_usuario".


du : Muestra el espacio que ocupan todos los directorios a partir del directorio actual.

  • du [opciones]
  • du -s : Nos muestra únicamente el total.
  • du -a : Muestra lo que ocupan los archivos, además de los directorios.


nano : Se utiliza para crear y editar un fichero.

uniq : Excluye todos los grupos de líneas adyacentes duplicados menos uno.

  • uniq [opciones] [nombre_archivo]
  • uniq -d [nombre_archivo] : Imprime solo líneas duplicadas.
  • uniq -u [nombre_archivo] : Elimina las líneas consecutivas iguales.
  • uniq -c [nombre_archivo] : Cuenta el número de ocurrencias de cada línea.
  • uniq -i [nombre_archivo] : Considera como igual las mayúsculas y minúsculas.



Comandos Intermedios



grep : Imprime las líneas (strings) que contengan una palabra en cierto archivo.

  • grep [opciones] [patrón] [nombre_archivo]
  • grep -c [patrón] [nombre_archivo] : En lugar de imprimir las líneas coincidentes, imprime la cantidad de líneas que coincidieron en cada archivo.
  • grep -e [patrón] [nombre_archivo] : Permite especificar que lo que viene a continuación es una expresión (y no una opción).
  • grep -f [nombre_archivo] : Obtiene los patrones del archivo.
  • grep -v [patrón] [nombre_archivo] : Muestra las líneas que no contienen el patrón.
  • grep -l [patrón] [nombre_archivo] : Muestra los archivos que contienen el patrón en alguna línea, en lugar de las líneas.
  • grep -H [patrón] [nombre_archivo] : Imprime el nombre del archivo con cada coincidencia.


sort : Se utiliza para ordenar líneas de textos a partir de varios criterios.

  • sort [opciones] [nombre_archivo]
  • sort -b [nombre_archivo] : Ignora los blancos (espacio, tabuladores, entre otros...) al principio de las líneas.
  • sort -c [nombre_archivo] : Revisa si los archivos están ordenados, pero no ordena.
  • sort -d [nombre_archivo] : Ignora los caracteres que no sean letras o números.
  • sort -f [nombre_archivo] : Ignora la diferencia entre mayúsculas y minúsculas.
  • sort -n [nombre_archivo] : Considera por criterio numérico.
  • sort -o [nombre_archivo] : Escribe el resultado en un archivo en vez de la salida estándar.
  • sort -r [nombre_archivo] : Invierte el orden de ordenamiento.
  • sort -u [nombre_archivo] : Si se encuentran varias líneas iguales, éstas sólo se reproducirán una vez.
  • sort -t [nombre_separador] : Especifica el carácter de separación entre campos. Por defecto es el espaciado.


strings : Muestra la cadena de caracteres imprimibles que tengan los archivos (4 Caracteres min.).

  • strings [opciones] [nombre_archivo]
  • strings -a [nombre_archivo] : Busca en todos los contenidos de los ficheros.
  • strings -f [nombre_archivo] : Muestra el nombre del fichero antes de cada cadena.
  • strings -n [min] / bytes [min] : Muestra secuencias de caracteres de al menos [min] caracteres de largo.


base64 : Codifica o decodifica una entrada o salida estándar.

  • base64 [opciones] [nombre_archivo]
  • base64 -e [nombre_archivo] : Codifica cualquier dato de una salida estándar o archivo. Opción por defecto.
  • base64 -d [nombre_archivo] : Decodifica cualquier dato de una salida estándar o archivo.
  • base64 -n [nombre_archivo] : Por defecto "base64" verifica errores mientras decodifica, con esta opción se ignora la verificación al momento de decodificar.
  • base64 -u [nombre_archivo] : Obtiene información sobre el comando.
  • base64 -i [nombre_archivo] : Ignora caracteres no alfabéticos mientras decodifica.
  • base64 -copyright [nombre_archivo] : Consigue información copyright.


tr : Asocia cada carácter del conjunto 1 con el conjunto 2, salvo que la operación no requiera este último conjunto.

  • tr [opciones] [conjunto_1] [conjunto_2] [nombre_archivo]
  • tr -c [conjunto_1] [conjunto_2] [nombre_archivo] : Define al [conjunto_1] como todos los caracteres que no se encuentran en la definición dada por el usuario.
  • tr -d [conjunto_1] [conjunto_2] [nombre_archivo] : Borra los caracteres de definidos en el [conjunto_1].
  • tr -s [conjunto_1] [conjunto_2] [nombre_archivo] : Elimina la secuencia continua de caracteres repetidos, definidos en el [conjunto_1].


tar : Puedes comprimir y descomprimir archivos .tar , no altera las características de los archivos y directorios.

  • tar [opciones] [nombre_archivo]
  • tar -c [nombre_archivo] : Crea un nuevo archivo .tar.
  • tar -v [nombre_archivo] : Muestra una descripción detallada del progreso de la compresión.
  • tar -r [nombre_archivo] : Agregar tanto archivo como directorio a un fichero comprimido (No se puede agregar .targz o .tar.bz2).


gzip : Comprime y descomprime archivos .gz.

  • gzip [opciones] [nombre_archivo]
  • gzip -d [nombre_archivo] : Descomprime archivos .gz.
  • gzip -v [nombre_archivo] : Muestra una descripción del progreso. Por defecto es compresión
  • gzip -k [nombre_archivo] : Mantiene ambas copias de la ejecución del archivo comprimido y descomprimido.
  • gzip -c [nombre_archivo] : Imprime la salida de un archivo comprimido antes de descomprimirlo [Tamaño comprimido, Tamaño sin comprimir, Relación compresión, Nombre no comprimido]
  • gzip -r [nombre_archivo] : Descomprime todos los archivos dentro de un escritorio.
  • gzip -t [nombre_archivo] : Comprueba si un archivo comprimido con gzip es valido.


bzip2 : Se utiliza para comprimir y descomprimir archivos .bz2.

  • bzip2 [opciones] [nombre_archivo]
  • bzip2 -d [nombre_archivo] : Fuerza la descompresión.
  • bzip2 -z [nombre_archivo] : Fuerza la compresión.
  • bzip2 -t [nombre_archivo] : Realiza una prueba de descompresión y muestra los resultados.
  • bzip2 -f [nombre_archivo] : Fuerza la sobreescritura del archivo de salida.
  • bzip2 -k [nombre_archivo] : Mantiene el archivo original.
  • bzip2 -q [nombre_archivo] : Suprime los mensajes de alerta no esenciales.
  • bzip2 -d [nombre_archivo] : Descomprime archivos .gz.


xxd : Crea un volcado hexadecimal de un archivo también puede transformarlo a su forma binaria inicial.

  • xxd [opciones] [nombre_archivo]
  • xxd -p [nombre_archivo] : Muestra el archivo en hexadecimal plano.
  • xxd -s [n°_lineas] [nombre_archivo] : Permite saltar un determinado [n°_líneas] al momento de crear un hexdump
  • xxd -l [n°_lineas] [nombre_archivo] : Limita las líneas al momento de crear el hexdump.
  • xxd -r [nombre_archivo] : Transforma de hexadecimal a binario.
  • xxd -p [n°_columnas] [nombre_archivo] : Permite elegir el determinado [n°_columnas] que tendrá el hexdump.
  • xxd -b [nombre_archivo] : Crea un volcado en digitos binarios.


mkdir : Crea carpetas (Directorios) si es que no existe uno con el nombre dado.

  • mkdir [opciones] [nombre_directorio]
  • mkdir -p [nombre_directorio] : Crea un directorio dentro de otro.
  • mkdir -m [nombre_directorio] : Establece los permisos (como chmod).


cp : Se utilizar para copiar archivos.

  • cp [opciones] [origen] [destino]
  • cp -f [origen] [destino] : Borra los archivos de destino ya existentes.
  • cp -p [origen] [destino] : Preserva los permisos, el usuario y el grupo del archivo a copiar.
  • cp -b [origen] [destino] : Crea una copia de seguridad de los archivos que sobrescribe.
  • cp -R [origen] [destino] : Copia los directorios recursivamente.
  • cp -u [origen] [destino] : No sobrescribe ficheros a menos que el origen sea más moderno que el destino
  • cp -v [origen] [destino] : Da información en pantalla sobre los archivos que se van copiando.


mv : Se usa para mover ficheros tanto como para renombrarlos.

  • mv [opciones] [nombre_archivo1] [nombre_archivo2] (para cambiar nombre)
  • mv [opciones] [nombre_archivo] [nombre_directorio] (para mover el fichero)
  • mv -i [nombre_archivo] [nombre_directorio] : Pregunta antes de sobrescribir un fichero existente en el directorio de destino.
  • mv -b [nombre_archivo] [nombre_directorio] : Crea una copia de seguridad de los archivos que sobrescribe.
  • mv -v [nombre_archivo] [nombre_directorio] : Muestra el nombre de cada archivo a ser movido.
  • mv -f [nombre_archivo] [nombre_directorio] : Borra los archivos de destino existente sin preguntar al usuario.


diff : Compara y muestra las diferencias entre archivos.

  • diff [nombre_archivo1] [nombre_archivo2]
  • diff -b [nombre_archivo1] [nombre_archivo2] : Indica que los espacios en blanco son irrelevantes para la comparación del contenido de las líneas.
