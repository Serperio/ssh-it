Instalar linux

  • ssh -i ["nombre_key"] ["nombre_usuario@dominio"] : Se utiliza para autentificar una key pública.

Crear usuarios/grupos

  • grep [opciones] [patrón] [nombre_archivo]
  • grep -c [patrón] [nombre_archivo] : En lugar de imprimir las líneas coincidentes, imprime la cantidad de líneas que coincidieron en cada archivo.
  • grep -e [patrón] [nombre_archivo] : Permite especificar que lo que viene a continuación es una expresión (y no una opción).
  • grep -f [nombre_archivo] : Obtiene los patrones del archivo.
  • grep -v [patrón] [nombre_archivo] : Muestra las líneas que no contienen el patrón.
  • grep -l [patrón] [nombre_archivo] : Muestra los archivos que contienen el patrón en alguna línea, en lugar de las líneas.
  • grep -H [patrón] [nombre_archivo] : Imprime el nombre del archivo con cada coincidencia.


Modificar usuario/grupo

  • sort [opciones] [nombre_archivo]
  • sort -b [nombre_archivo] : Ignora los blancos (espacio, tabuladores, entre otros...) al principio de las líneas.
  • sort -c [nombre_archivo] : Revisa si los archivos están ordenados, pero no ordena.
  • sort -d [nombre_archivo] : Ignora los caracteres que no sean letras o números.
  • sort -f [nombre_archivo] : Ignora la diferencia entre mayúsculas y minúsculas.
  • sort -n [nombre_archivo] : Considera por criterio numérico.
  • sort -o [nombre_archivo] : Escribe el resultado en un archivo en vez de la salida estándar.
  • sort -r [nombre_archivo] : Invierte el orden de ordenamiento.
  • sort -u [nombre_archivo] : Si se encuentran varias líneas iguales, éstas sólo se reproducirán una vez.
  • sort -t [nombre_separador] : Especifica el carácter de separación entre campos. Por defecto es el espaciado.


Eliminar usuario/grupo

  • strings [opciones] [nombre_archivo]
  • strings -a [nombre_archivo] : Busca en todos los contenidos de los ficheros.
  • strings -f [nombre_archivo] : Muestra el nombre del fichero antes de cada cadena.
  • strings -n [min] / bytes [min] : Muestra secuencias de caracteres de al menos [min] caracteres de largo.


Permisos

  • base64 [opciones] [nombre_archivo]
  • base64 -e [nombre_archivo] : Codifica cualquier dato de una salida estándar o archivo. Opción por defecto.
  • base64 -d [nombre_archivo] : Decodifica cualquier dato de una salida estándar o archivo.
  • base64 -n [nombre_archivo] : Por defecto "base64" verifica errores mientras decodifica, con esta opción se ignora la verificación al momento de decodificar.
  • base64 -u [nombre_archivo] : Obtiene información sobre el comando.
  • base64 -i [nombre_archivo] : Ignora caracteres no alfabéticos mientras decodifica.
  • base64 -copyright [nombre_archivo] : Consigue información copyright.


Permisos especiales

  • tr [opciones] [conjunto_1] [conjunto_2] [nombre_archivo]
  • tr -c [conjunto_1] [conjunto_2] [nombre_archivo] : Define al [conjunto_1] como todos los caracteres que no se encuentran en la definición dada por el usuario.
  • tr -d [conjunto_1] [conjunto_2] [nombre_archivo] : Borra los caracteres de definidos en el [conjunto_1].
  • tr -s [conjunto_1] [conjunto_2] [nombre_archivo] : Elimina la secuencia continua de caracteres repetidos, definidos en el [conjunto_1].


Puertos

  • tar [opciones] [nombre_archivo]
  • tar -c [nombre_archivo] : Crea un nuevo archivo .tar.
  • tar -v [nombre_archivo] : Muestra una descripción detallada del progreso de la compresión.
  • tar -r [nombre_archivo] : Agregar tanto archivo como directorio a un fichero comprimido (No se puede agregar .targz o .tar.bz2).
